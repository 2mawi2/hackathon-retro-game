# Retro Game Jam: Go Nuts!

A retro-style 2D action game built with TypeScript and HTML5 Canvas.

## Play

```bash
npm install
npm run dev
```

## Controls

| Player 1 | Player 2 |
|----------|----------|
| WASD - Move | Arrows - Move |
| Space - Attack | Enter - Attack |

**T** - Skill Tree | **S** - Shop | **ESC** - Pause

## Features

- 7 themed levels with unique mechanics
- 2-player local co-op
- Equipment upgrades & skill tree progression
- Multiple enemy types and boss battles
